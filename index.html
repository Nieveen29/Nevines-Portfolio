Covid Project in SQL

In this project, I used SQL to analyze global COVID-19 trends, focusing on case counts, death rates, and vaccination progress. I examined the likelihood of death after infection, calculated the percentage of the population infected in each country, and tracked changes over time. By joining datasets and using window functions, I monitored rolling vaccination numbers and identified countries and continents most impacted by the pandemic. The project involved building temporary tables and views to organize the data for future visualizations.

Nashville Housing Data Cleaning in SQL

In this project, I cleaned a Nashville housing dataset using SQL to make it analysis-ready. I standardized date formats, filled in missing property addresses by referencing Parcel IDs, and split compound address fields—such as property and owner addresses—into separate columns for street, city, and state. I also standardized inconsistent values in the "Sold as Vacant" field and removed duplicate records using SQL window functions. Finally, I dropped unnecessary columns to streamline the dataset.
